INDRA

INDonesian Resource GrAmmar

INDRA

  • an open-source computational grammar for Indonesian.
  • modelled in Head-Driven Phrase Structure Grammar (HPSG)
    and Minimal Recursion Semantics (MRS).
  • created and developed using tools from DELPH-IN.
  • aims to parse Indonesian text in NTU-MC.
  • will be applied to machine translation.
  • stored in GitHub.

Theoretical framework

Head-Driven Phrase Structure Grammar (HPSG)

  • Phrase Structure Grammar:
    • a set of rules:
      S      → NP VP
      NP → N
      VP → V
    • a lexicon of POS and words:
      N: anjing "dog"
      V: menggonggong "bark"
  • What sentence(s) can be generated from the rules and lexicon above?
    • Anjing menggonggong "Dogs bark"
  • HPSG adds the idea that the lexical head is passed up the parse tree.

(Sag et al., 2003; Jurafsky & Martin, 2009)
